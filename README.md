# acad-web-app
acad-web-app Demo

## make directories to hold files
1. ./__tempFiles__
2. ./public/__pdf__

## Install dependencies
````bash
npm install
````

## start mongo db
````bash
sudo mongod service start
````

## run

````bash
node app.js
````
or use nodemon 

````bash
nodemon
````

## entrypoint 
http://localhost:3000/edit/allCourses
